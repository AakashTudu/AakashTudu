### 🏗️ Core Engineering Portfolio

**1. Applied Cryptography & zkML: [Aegis Core (zkCredit)](https://github.com/AakashTudu/aegis-core)**
A fully on-chain, Zero-Knowledge credit oracle. Bypasses traditional overcollateralization by natively un-wrapping finite-field ML predictions on-chain to dynamically route undercollateralized liquidity. 
* *Live on Sepolia Testnet.*

**2. Smart Contract Engineering: [EVM Mechanics & Optimization](https://github.com/AakashTudu/evm-mechanics-and-optimization)**
An applied research repository demonstrating deep EVM state manipulation. Features mathematical proofs of gas reductions using explicit bitwise storage packing (~58% execution savings) and Inline Assembly/Yul opcodes to bypass Solidity compiler overhead.
* *Built & fuzzed entirely in Foundry.*

**3. DeFi Risk Analytics: [Systemic Risk Engine](https://github.com/AakashTudu/defi-systemic-risk-engine)**
An interactive simulation engine modeling cascading liquidations within Aave V3. Replaces basic API fetching with probabilistic Data Science models (log-normal wealth distribution, Beta LTV modeling) to stress-test liquidity pools against extreme market crashes.
* *Built with Python, Pandas, and Streamlit.*
